# RL Discord Status

Shows your Rocket League game info on your Discord status in real-time.

**Displays:**
- Stadium name
- Score (your team first)
- Time remaining (updates every ~2 seconds)
- Game mode
- Your rank (ranked playlists)
- Searching for Match / In Menus states

## Installation

Install via the [BakkesMod Plugin Manager](https://bakkesmodplugins.com/plugins/RLDiscordStatus).

The plugin installer will handle everything automatically.

## Settings

**F2 → Plugins → RL Discord Status**

- Show/hide stadium, score, time, game mode, rank, elapsed timer
- Score style: Full (`Blue 2 - 1 Orange`) / Minimal (`2 - 1`) / Context (`Winning 2-1`)

## Requirements

- [BakkesMod](https://bakkesmod.com)
- Discord running
